Before installing a release-qualified chart:

1. label the dedicated namespace for the privileged Pod Security level;
2. create the external installation identity and Scaleway credential Secrets;
3. verify the frozen CSI driver name, supply real Project/parent IDs, and pin
   every release image by tag plus immutable sha256 digest;
4. verify every schedulable Linux node is compatible and can attach every
   configured parent;
5. run the documented preflight.

Do not run `helm uninstall` directly. First complete and verify the matching
version of `csi-admin uninstall prepare`. Helm deliberately does not own the
runtime leadership Lease, approval/checkpoint Secrets, allocation tombstones,
parent claims, or user data.
